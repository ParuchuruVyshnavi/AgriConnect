# 🌾 AgriConnect

A Smart Agriculture Resource Sharing Platform designed to help farmers access equipment, labour, water resources, crop-loss reporting, and agricultural marketplaces through a single platform.

## 🚀 Features

### 🏠 Home
- Modern responsive landing page
- Agriculture updates
- Testimonials
- Statistics dashboard

### 🚜 Equipment Rental
- Browse farming equipment
- Equipment categories
- Rental request forms
- Equipment listings

### 👨‍🌾 Labour Connect
- Find agricultural jobs
- Hire skilled workers
- Post job opportunities
- Worker profiles

### 🛒 Marketplace
- Buy & sell agricultural products
- Crop waste exchange
- Product listings
- Product submission forms

### 💧 Water Sharing
- Water resource sharing
- Irrigation support requests
- Water availability tracking

### 🌾 Crop Loss Reporting
- Report crop damage
- Loss assessment forms
- Farmer support system

### 📚 Knowledge Hub
- Farming guides
- Agricultural resources
- Learning materials

### 📞 Contact
- Contact forms
- Support information
